AIMMDB Project

Welcome to the AIMMDB project! This repository contains data sets from the AIMM Project.

Data Sets

The AIMM Database is a repository of data sets collected by the AIMM Project. The datasets are managed by the Bluesky Tiled data access server. They can be access by the AIMMDB Tiled web page, as well as directly from python code by using the Tiled Client.
